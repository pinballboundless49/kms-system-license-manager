### Activating New Licenses

1. Click **"Activate"** button
2. Select product type: Windows or Office
3. Tool automatically detects installed products
4. Click **"Activate Selected"**
5. Choose activation method:
   - **HWID** (Windows 10/11) — permanent
   - **KMS** (all products) — 180 days
   - **OEM** (branded PCs)
6. Click **"Confirm Activation"**
7. License added to inventory with 180-day countdown
8. Auto-renewal service configured automatically

### Bulk Activation (Multiple Products)

1. Click **"Activate"** → **"Bulk Mode"**
2. Select all products to activate:
   ```
   ☑  Windows 11 Pro
   ☑  Windows 10 (dual boot)
   ☑  Office 2024 Professional Plus
   ☑  Office 2021 Standard
   ☐  Visio 2024 (not installed)
   ☐  Project 2024 (not installed)
   ```
3. Click **"Activate All Selected"**
4. Progress bar shows each activation
5. Summary report generated:
   ```
   Bulk Activation Complete
   ────────────────────────
   Successful: 4
   Failed: 0
   Skipped: 2 (not installed)
   Duration: 18 seconds
   ```
6. Click **"Save Report"** for records

### Renewal Management

#### Manual Renewal

1. Click **"Renew"** button
2. Select expiring license(s)
3. Click **"Renew Selected"**
4. Tool re-activates using same method
5. Expiration resets to 180 days
6. Log entry created

#### Automatic Renewal Setup

1. Click **"Settings"** → **"Auto-Renewal"**
2. Enable **"Automatic License Renewal"**
3. Set renewal threshold (default: 30 days before expiry)
4. Choose renewal time (default: 2:00 AM)
5. Click **"Save Settings"**
6. Service runs silently in background
7. Never worry about expired licenses again

### Email Alert Configuration

1. Click **"Alerts"** button
2. Enable **"Send Email Alerts"**
3. Configure SMTP settings:
   ```
   SMTP Server: smtp.gmail.com
   Port: 587
   Encryption: TLS
   Username: admin@yourcompany.com
   Password: ••••••••
   ```
4. Set alert thresholds:
   - ☑ 30 days before expiry (warning)
   - ☑ 15 days before expiry (urgent)
   - ☑ 7 days before expiry (critical)
   - ☑ 1 day before expiry (final notice)
   - ☑ On successful renewal
   - ☑ On activation failure
5. Add recipient email addresses
6. Click **"Send Test Alert"**
7. Click **"Save Configuration"**

### Backup Scheduler

1. Click **"Settings"** → **"Backup"**
2. Enable **"Automatic License Backups"**
3. Set backup frequency:
   - Daily (recommended)
   - Weekly
   - Monthly
4. Choose backup location:
   - Local folder
   - Network share
   - External drive
5. Set retention policy:
   - Keep last 7 backups (default)
   - Keep last 30 backups
   - Keep all backups
6. Click **"Save Settings"**
7. Backups run automatically on schedule
